Q1.  What is a comprehension in Python? What is the main
     advantage over a traditional for-loop?
     Comprehension are a readable way to create list,set and dictionary by iterating over a sequence and optionally applying a condition
     main advantage is we written comprehension in single line 

Q2.  What is the general syntax of a list comprehension?
     Explain each part: expression, for clause, if condition.
     syntax : [expression for item in iterable if condition]
     expression : the expression we want to perform on list
     for clause : its a iterable sequence using for loop
     if condition : its optional used when some conditions are apply

Q13. How do you filter key-value pairs in a dictionary
     comprehension? Give an example.

     using if condition we filter key-value pair
     for ex;
     prices = {'apple':30, 'banana':10, 'mango':80, 'grape':120}
     filtered = {k:v for k,v in prices.items() if v > 50}
    print(filtered)

Q17. Can you use a function call inside the expression part
     of a comprehension? Give an example.
     Yes, we can used function call inside the expression part of a comprehension
     for ex;
     words = ['python', 'java', 'c++']
     upper_words = [word.upper() for word in words]
